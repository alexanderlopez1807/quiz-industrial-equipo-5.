INSTRUCCIONES DE USO DEL QUIZ

Esta aplicación es un Quiz interactivo desarrollado en Streamlit cuyo objetivo es evaluar conocimientos básicos 

ENLACE: 
https://quiz-industrial-equipo-5.streamlit.app/

Como utilizar la aplicacion:

Ingresar al link:
- te aparecera una pantalla de inicio donde podras comenzar
- lee las instrucciones antes de iniciar

Registra tu nombre:
- para iniciar con las preguntas es importante poner tu nombre 
- este quedara registrado al final con su calificacion

Responder las preguntas: 
- se mostraran 4 preguntas aleatorias, una por una
- en cada pregunta selecciona el inciso que consideres correcto
- cuando ya tengas la respuesta preciona "responder" para avanzar

Resultados:
- la aplicacion automaticamente evaluara tus respuestas 
- al finalizar el quiz tus resultados se daran 
- tu nombre y puntaje se quedan registrados para compararlos con otras personas

Volver a jugar:
  - si deseas volver a inicar da click en "reiniciar juego"
  - se reiniciaran todas las respuestas y apareceran nuevas preguntas
